# Lecture Notes Symbol Index

This index lists symbols used across the lecture-note chapters. Meanings and conventions are written to remain consistent with the book. The table is sorted alphabetically by the symbol as written (case-insensitive).




| Symbol                | Meaning                                 | Description                                                                                                                                                                                | Value                         | Units        |
| --------------------- | --------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ----------------------------- | ------------ |
| $A$                   | Area                                    | Electrode area or effective area used in capacitor models such as $C=\varepsilon\varepsilon_0A/d$.                                                                                         |                               | m$^{2}$      |
| $a_i^\alpha$          | Activity                                | Dimensionless thermodynamic measure of the effective concentration of species $i$ in phase $\alpha$, defined through $\mu_i^\alpha=\mu_i^{\circ,\alpha}+RT\ln a_i^\alpha$.                 |                               | —            |
| $C$                   | Capacitance                             | Capacitance in the capacitor relation $q=CE$.                                                                                                                                              |                               | F            |
| $C(x,t)$              | Concentration                           | Concentration profile as a function of distance from the electrode and time (e.g., the one-dimensional boundary-layer field used in diffusion-layer sketches).                               |                               | mol m$^{-3}$ |
| $C(x=0)$              | Surface concentration                    | Concentration of a dissolved species evaluated at the electrode surface ($x=0$).                                                                                                            |                               | mol m$^{-3}$ |
| $C^{*}$               | Bulk concentration                       | Bulk (far-from-electrode) concentration of a dissolved species, treated as constant under the diffusion-layer approximation.                                                               |                               | mol m$^{-3}$ |
| $C_{\mathrm{O}}^{*}$  | Bulk oxidized-form concentration         | Bulk concentration of the oxidized form $\mathrm{O}$ used in steady-state limiting-current expressions such as $i_{\mathrm{L,c}}=nFA\,m_{\mathrm{O}}\,C_{\mathrm{O}}^{*}$.               |                               | mol m$^{-3}$ |
| $C_{\mathrm{O}}(x=0)$ | Surface oxidized-form concentration      | Surface concentration of the oxidized form $\mathrm{O}$ at $x=0$ used in Nernst and limiting-current discussions.                                                                          |                               | mol m$^{-3}$ |
| $C_{\mathrm{R}}^{*}$  | Bulk reduced-form concentration          | Bulk concentration of the reduced form $\mathrm{R}$; often taken as $C_{\mathrm{R}}^{*}\approx 0$ for a cathodic wave starting from an oxidized bulk.                                     |                               | mol m$^{-3}$ |
| $C_{\mathrm{R}}(x=0)$ | Surface reduced-form concentration       | Surface concentration of the reduced form $\mathrm{R}$ at $x=0$ used in Nernst and limiting-current discussions.                                                                           |                               | mol m$^{-3}$ |
| $C_{\mathrm{d}}$      | Double-layer capacitance                | Capacitance associated with the electrode/solution double layer; governs charging transients and capacitive currents in non-faradaic windows.                                              |                               | F            |
| $C_{\mathrm{SCE}}$    | Reference-electrode capacitance (model) | Effective capacitance associated with a reference electrode (here, SCE) in the lecture-note equivalent-circuit model; combined in series with $C_{\mathrm{d}}$ to form $C_{\mathrm{tot}}$. |                               | F            |
| $C_{\mathrm{tot}}$    | Total series capacitance                | Effective capacitance seen by the external circuit when $C_{\mathrm{d}}$ and $C_{\mathrm{SCE}}$ are in series: $1/C_{\mathrm{tot}}=1/C_{\mathrm{d}}+1/C_{\mathrm{SCE}}$.                   |                               | F            |
| $d$                   | Separation distance                     | Effective separation distance in the parallel-plate capacitance approximation.                                                                                                             |                               | m            |
| $D$                   | Diffusion coefficient                    | Diffusion coefficient appearing in Fick’s law and diffusion-layer (film) approximations for transport to an electrode.                                                                     |                               | m$^{2}$ s$^{-1}$ |
| $\delta$              | Diffusion-layer thickness                | Effective diffusion-layer thickness used in the film approximation to represent the near-electrode concentration boundary layer.                                                           |                               | m            |
| $\Delta E$            | Potential-step magnitude                | Magnitude of an applied voltage step in the charging-transient derivations.                                                                                                                |                               | V            |
| $\Delta G$            | Gibbs free energy change                | Gibbs free energy change (typically for a specified process or reaction direction); for reversible cell operation, $\Delta G=-nF E_{\text{cell}}$.                                           |                               | J mol$^{-1}$ |
| $\Delta G_r$          | Gibbs free energy of reaction           | Gibbs free energy change per mole of reaction as written; for reversible cell operation, $\Delta G_r=-nF E_{\text{cell}}$.                                                                  |                               | J mol$^{-1}$ |
| $\Delta G_r^\circ$    | Standard Gibbs free energy of reaction  | Standard-state Gibbs free energy change per mole of reaction as written; related to standard cell potential by $\Delta G_r^\circ=-nF E_{\text{cell}}^\circ$.                               |                               | J mol$^{-1}$ |
| $E$                   | Electrode potential                     | Electrode potential of a metal/solution interface defined (by convention) as $E=\phi^{\mathrm{M}}-\phi^{\mathrm{S}}$; also used as a generic applied/measured potential difference in equivalent-circuit and voltammetry contexts.   |                               | V            |
| $E_{1/2}$             | Half-wave potential                     | Potential on a steady-state sigmoidal wave at which $i=i_{\mathrm{L,c}}/2$ (for a cathodic wave); equals $E^{0}$ only when the relevant mass-transfer coefficients are equal.               |                               | V            |
| $E_{C_{\mathrm{d}}}$  | Capacitor (double-layer) voltage        | Voltage drop across $C_{\mathrm{d}}$, equal to $q/C_{\mathrm{d}}$ in the capacitor model.                                                                                                  |                               | V            |
| $E_{R_{\mathrm{s}}}$  | Ohmic (solution) drop                   | Voltage drop across the solution resistance $R_{\mathrm{s}}$, equal to $iR_{\mathrm{s}}$.                                                                                                  |                               | V            |
| $E_{\text{anode}}$    | Anode potential                         | Electrode potential of the anode (the oxidation electrode in the spontaneous cell direction) used in the convention $E_{\text{cell}}=E_{\text{cathode}}-E_{\text{anode}}$.                |                               | V            |
| $E_{\text{cathode}}$  | Cathode potential                       | Electrode potential of the cathode (the reduction electrode in the spontaneous cell direction) used in the convention $E_{\text{cell}}=E_{\text{cathode}}-E_{\text{anode}}$.             |                               | V            |
| $E_{\text{cell}}$     | Cell potential                          | Cell potential defined (by convention) as $E_{\text{cell}}=E_{\text{cathode}}-E_{\text{anode}}$ and related to spontaneity by $\Delta G=-nF E_{\text{cell}}$.                             |                               | V            |
| $E_{\text{cell}}^\circ$ | Standard cell potential                 | Cell potential under the chosen standard-state conditions; related to standard reaction free energy by $\Delta G_r^\circ=-nF E_{\text{cell}}^\circ$.                                       |                               | V            |
| $E^{0}$               | Standard potential (alternate notation) | Alternate notation for the standard potential used in some lecture-note chapters; treated as synonymous with $E^\circ$ in Nernst-type relations.                                            |                               | V            |
| $E^\circ$             | Standard potential                      | Standard (reference) potential under the stated standard-state conventions; appears as the constant term in Nernst-type relations.                                                          |                               | V            |
| $\varepsilon$         | Relative permittivity                   | Relative permittivity (dielectric constant) in the parallel-plate capacitance model.                                                                                                       |                               | —            |
| $\varepsilon_{0}$     | Vacuum permittivity                     | Permittivity of free space in the parallel-plate capacitance expression.                                                                                                                   | $8.8541878128\times 10^{-12}$ | F m$^{-1}$   |
| $F$                   | Faraday constant                        | Charge per mole of electrons, appearing in Faraday’s law.                                                                                                                                  | 96485                         | C mol$^{-1}$ |
| $i$                   | Current                                 | Electrical current; in a non-faradaic window, dominated by charging current.                                                                                                               |                               | A            |
| $i(0^+)$              | Initial current                         | Initial (post-step) charging current in a voltage-step experiment, typically $i(0^+)=\Delta E/R_{\mathrm{s}}$ for the simple series model.                                                 |                               | A            |
| $i_{\mathrm{C}}$      | Capacitive (charging) current           | Capacitive current defined by $i_{\mathrm{C}}=dq/dt=C\,dE/dt$ under the capacitor model.                                                                                                   |                               | A            |
| $i_{\mathrm{L}}$      | Limiting current                         | Plateau magnitude of faradaic current under transport control (e.g., diffusion-limited conditions).                                                                                        |                               | A            |
| $i_{\mathrm{L,c}}$    | Cathodic limiting current               | Transport-limited plateau current for a cathodic (reduction) wave; in the film approximation, $i_{\mathrm{L,c}}=nFA\,m_{\mathrm{O}}\,C_{\mathrm{O}}^{*}$.                                |                               | A            |
| $J$                   | Flux (to electrode, magnitude)          | Molar flux of a species to the electrode surface (used as a nonnegative magnitude when relating transport supply to current).                                                              |                               | mol m$^{-2}$ s$^{-1}$ |
| $J_x$                 | Flux component (in $+x$ direction)      | Component of the molar flux in the $+x$ direction; in 1D diffusion, $J_x=-D\,dC/dx$.                                                                                                       |                               | mol m$^{-2}$ s$^{-1}$ |
| $k^{0}$               | Standard rate constant                  | Kinetic parameter that sets the intrinsic speed of interfacial electron transfer under standard-state conditions; larger $k^{0}$ corresponds to faster electron-transfer kinetics.           |                               | m s$^{-1}$   |
| $m$                   | Mass-transfer coefficient               | Lumped transport parameter relating flux to a concentration difference, often $m=D/\delta$ in the diffusion-layer (film) approximation.                                                    |                               | m s$^{-1}$   |
| $m_{\mathrm{O}}$      | Mass-transfer coefficient (oxidized)    | Mass-transfer coefficient for the oxidized form $\mathrm{O}$ in steady-state film relations such as $i_{\mathrm{L,c}}=nFA\,m_{\mathrm{O}}\,C_{\mathrm{O}}^{*}$.                          |                               | m s$^{-1}$   |
| $m_{\mathrm{R}}$      | Mass-transfer coefficient (reduced)     | Mass-transfer coefficient for the reduced form $\mathrm{R}$ in steady-state film relations such as $C_{\mathrm{R}}(x=0)=i/(nFA\,m_{\mathrm{R}})$.                                        |                               | m s$^{-1}$   |
| $N(e^-)$              | Number of electrons                     | Number of electrons corresponding to a passed charge, $N(e^-)=n(e^-)N_{\mathrm{A}}$.                                                                                                       |                               | —            |
| $N_{\mathrm{A}}$      | Avogadro constant                       | Conversion factor between moles and particle count.                                                                                                                                        | $6.02214076\times 10^{23}$    | mol$^{-1}$   |
| $n$                   | Electron stoichiometric number          | Number of electrons transferred per mole of overall reaction as written; appears in $E_{\text{cell}}=E_{\text{cell}}^\circ-(RT/nF)\ln\mathcal{Q}$ and $\Delta G_r=-nF E_{\text{cell}}$. |                               | —            |
| $n(e^-)$              | Moles of electrons                      | Moles of electrons corresponding to passed charge, $n(e^-)=Q/F$.                                                                                                                           |                               | mol          |
| $\mathrm{O}$          | Oxidized form                           | Generic oxidized species in an $\mathrm{O}/\mathrm{R}$ couple written $\mathrm{O}+ne^- \rightleftharpoons \mathrm{R}$.                                                                       |                               | —            |
| $\mathcal{Q}$         | Reaction quotient                       | Activity product for a net cell reaction, $\mathcal{Q}=\prod_k a_k^{\nu_k}$; enters the cell Nernst equation $E_{\text{cell}}=E_{\text{cell}}^\circ-(RT/nF)\ln\mathcal{Q}$.              |                               | —            |
| $Q$                   | Charge passed                           | Total charge passed through an external circuit.                                                                                                                                           |                               | C            |
| $q$                   | Stored charge                           | Charge stored on a capacitor (e.g., the double layer), related by $q=C_{\mathrm{d}}E_{C_{\mathrm{d}}}$.                                                                                    |                               | C            |
| $\mathrm{R}$          | Reduced form                            | Generic reduced species in an $\mathrm{O}/\mathrm{R}$ couple written $\mathrm{O}+ne^- \rightleftharpoons \mathrm{R}$.                                                                        |                               | —            |
| $R$                   | Gas constant                            | Universal gas constant appearing in activity-form chemical potentials and Nernst-type relations.                                                                                            | 8.314462618                   | J mol$^{-1}$ K$^{-1}$ |
| $R_{\mathrm{s}}$      | Solution resistance                     | Effective resistance of the electrolyte path; sets Ohmic drop and, with $C_{\mathrm{d}}$, the time constant $\tau=R_{\mathrm{s}}C_{\mathrm{d}}$.                                           |                               | $\Omega$     |
| $T$                   | Temperature                             | Absolute temperature in thermodynamic relations such as $\mu=\mu^\circ+RT\ln a$.                                                                                                            |                               | K            |
| $t$                   | Time                                    | Time variable in transient responses.                                                                                                                                                      |                               | s            |
| $v$                   | Scan rate                               | Slope of a linear potential ramp, $v=dE/dt$.                                                                                                                                               |                               | V s$^{-1}$   |
| $x$                   | Distance coordinate                     | Distance coordinate measured normal to a planar electrode surface, with $x=0$ at the electrode and $x>0$ extending into solution.                                                          |                               | m            |
| $x_{1}$               | IHP location (sketch coordinate)        | Distance coordinate marking the inner Helmholtz plane (IHP) location in the electric-double-layer sketch.                                                                                  |                               | m            |
| $x_{2}$               | OHP location (sketch coordinate)        | Distance coordinate marking the outer Helmholtz plane (OHP) location in the electric-double-layer sketch.                                                                                  |                               | m            |
| $z_i$                 | Charge number                           | Integer charge number of species $i$ (e.g., $z_{\mathrm{Zn}^{2+}}=+2$, $z_e=-1$), used in the electrochemical-potential term $z_iF\phi$.                                                   |                               | —            |
| $\bar{\mu}_i^\alpha$  | Electrochemical potential               | Molar free energy of species $i$ in phase $\alpha$ including electrical contributions, $\bar{\mu}_i^\alpha=\mu_i^\alpha+z_iF\phi^\alpha$.                                                  |                               | J mol$^{-1}$ |
| $\mu_e^{\mathrm{M}}$  | Electron chemical potential (metal)     | Chemical part of the electron’s molar free energy in a metal phase; appears in electrochemical-potential bookkeeping as $\bar{\mu}_e^{\mathrm{M}}=\mu_e^{\mathrm{M}}-F\phi^{\mathrm{M}}$. |                               | J mol$^{-1}$ |
| $\mu_i^\alpha$        | Chemical potential                      | Chemical contribution to the molar free energy of species $i$ in phase $\alpha$; for solutions, $\mu_i^\alpha=\mu_i^{\circ,\alpha}+RT\ln a_i^\alpha$.                                      |                               | J mol$^{-1}$ |
| $\mu_i^{\circ,\alpha}$ | Standard chemical potential             | Standard-state chemical potential of species $i$ in phase $\alpha$ appearing in $\mu_i^\alpha=\mu_i^{\circ,\alpha}+RT\ln a_i^\alpha$.                                                      |                               | J mol$^{-1}$ |
| $\nu_k$               | Stoichiometric coefficient              | Stoichiometric coefficient of species $k$ in a reaction written $\sum_k \nu_k A_k=0$, with $\nu_k>0$ for products and $\nu_k<0$ for reactants.                                             |                               | —            |
| $\phi$                | Galvani potential                       | Inner electrical potential of a phase; appears in electrochemical potentials as $z_iF\phi$ and in electrode potentials as a potential difference across an interface.                         |                               | V            |
| $\sigma^{\mathrm{d}}$ | Diffuse-layer charge density            | Diffuse-layer contribution to the solution-side charge density in the double layer.                                                                                                        |                               | C m$^{-2}$   |
| $\sigma^{\mathrm{i}}$ | Inner-layer charge density              | Inner-layer contribution to the solution-side charge density in the double layer.                                                                                                          |                               | C m$^{-2}$   |
| $\sigma^{\mathrm{m}}$ | Metal charge density                    | Excess charge density on the metal side of the interface.                                                                                                                                  |                               | C m$^{-2}$   |
| $\sigma^{\mathrm{S}}$ | Solution-side charge density            | Total solution-side charge density, $\sigma^{\mathrm{S}}=\sigma^{\mathrm{i}}+\sigma^{\mathrm{d}}=-\sigma^{\mathrm{m}}$.                                                                    |                               | C m$^{-2}$   |
| $\tau$                | Time constant                           | RC time constant; for the series $R_{\mathrm{s}}$–$C_{\mathrm{d}}$ model, $\tau=R_{\mathrm{s}}C_{\mathrm{d}}$.                                                                             |                               | s            |
