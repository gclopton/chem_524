# Chapter 1 Symbol Index

| Symbol | Meaning | Description | Value | Units |
| --- | --- | --- | --- | --- |
| $A$ | Area | Electrode area or plate overlap area in the parallel-plate capacitance model (Chapter 1 lecture notes; Eq. “Parallel-plate capacitance model”). |  | m$^{2}$ |
| $C$ | Capacitance | Capacitance in the capacitor relation $q=CE$ (Chapter 1 lecture notes; Eq. “Capacitor relation for interfacial charging”). |  | F |
| $C_{\mathrm{d}}$ | Double-layer capacitance | Capacitance associated with the electrode/solution double layer, used in RC charging expressions (e.g., Eq. (1.6.6), (1.6.10), (1.6.23)). |  | F |
| $d$ | Separation distance | Effective separation distance between “plates” in the parallel-plate capacitance approximation. |  | m |
| $E$ | Potential difference | Potential difference across an interfacial capacitor (lecture notes) or the magnitude of an applied potential step in the RC expressions (e.g., Eq. (1.6.6), (1.6.10)). |  | V |
| $\varepsilon$ | Relative permittivity | Relative permittivity (dielectric constant) of the material separating charge, appearing in $C=\varepsilon\varepsilon_0A/d$ (lecture notes). |  | — |
| $\varepsilon_{0}$ | Vacuum permittivity | Permittivity of free space in the parallel-plate capacitance expression. | $8.8541878128\times 10^{-12}$ | F m$^{-1}$ |
| $F$ | Faraday constant | Charge per mole of electrons in Faraday’s law (lecture notes). | 96485 | C mol$^{-1}$ |
| $i$ | Current | Electrical current (including charging/capacitive current in Chapter 1). |  | A |
| $N(e^-)$ | Number of electrons | Number of electrons corresponding to a passed charge, computed from $N(e^-)=n(e^-)N_{\mathrm{A}}$ (lecture notes). |  | — |
| $N_{\mathrm{A}}$ | Avogadro constant | Conversion factor from moles to particle count. | $6.02214076\times 10^{23}$ | mol$^{-1}$ |
| $n(e^-)$ | Moles of electrons | Moles of electrons corresponding to passed charge, $n(e^-)=Q/F$ (lecture notes). |  | mol |
| $Q$ | Charge passed | Total charge passed through a circuit or cell. |  | C |
| $q$ | Stored charge | Charge stored on a capacitor or in the double layer (e.g., Eq. (1.6.10); lecture notes Eq. $q=CE$). |  | C |
| $R_{\mathrm{s}}$ | Series resistance | Series resistance in a simple RC charging model (Eq. (1.6.6), (1.6.10)). |  | $\Omega$ |
| $R_{\mathrm{u}}$ | Uncompensated resistance | Uncompensated solution resistance in a three-electrode cell charging model (Eq. (1.6.23)). |  | $\Omega$ |
| $t$ | Time | Time variable in transient charging relations. |  | s |
| $v$ | Sweep rate | Linear potential sweep rate in Eq. (1.6.23). |  | V s$^{-1}$ |
| $x_{1}$ | IHP location (sketch coordinate) | Distance coordinate marking the inner Helmholtz plane (IHP) location in the lecture sketch (Figure 1.6a.3). |  | m |
| $x_{2}$ | OHP location (sketch coordinate) | Distance coordinate marking the outer Helmholtz plane (OHP) location in the lecture sketch (Figure 1.6a.3). |  | m |
| $\sigma^{\mathrm{d}}$ | Diffuse-layer charge density | Contribution of the diffuse layer to the solution-side charge density in the double layer (lecture notes Eq. “Double-layer charge-density bookkeeping”). |  | C m$^{-2}$ |
| $\sigma^{\mathrm{i}}$ | Inner-layer charge density | Contribution of the inner (compact) layer to the solution-side charge density in the double layer (lecture notes Eq. “Double-layer charge-density bookkeeping”). |  | C m$^{-2}$ |
| $\sigma^{\mathrm{m}}$ | Metal charge density | Excess charge density on the metal electrode side of the interface (lecture notes Eq. “Double-layer charge-density bookkeeping”). |  | C m$^{-2}$ |
| $\sigma^{\mathrm{S}}$ | Solution-side charge density | Total solution-side charge density, $\sigma^{\mathrm{S}}=\sigma^{\mathrm{i}}+\sigma^{\mathrm{d}}$ (lecture notes Eq. “Double-layer charge-density bookkeeping”). |  | C m$^{-2}$ |
